## 핵심 내러티브 (Core Narrative)

> "커머디티 트레이딩의 본질을 이해하는 세일즈맨이 데이터 플랫폼으로 무대를 옮긴다."

- 홍석진은 화학 B2B 트레이딩 3년차로, 커머디티 트레이딩의 전 과정을 실무에서 경험함.
- 동시에 Python과 AI 분석 툴을 독학하여 데이터 기반 영업 접근법을 구축함 — 이는 대부분의 영업 직원이 하지 않는 일.
- 트릿지는 전통적 트레이딩을 데이터와 기술로 혁신하는 회사. 홍석진의 프로필은 "트릿지가 필요로 하는 하이브리드 인재상"과 정확히 맞물림:
  - 전통 트레이딩 실전 경험 (화학 B2B) → 농식품 B2B로의 산업 전환이지만 트레이딩 본질은 동일
  - 데이터 분석 능력 (Python/AI) → 트릿지의 데이터 기반 컨설팅 세일즈와 즉시 시너지
  - 영어 + 글로벌 바이어 경험 → 트릿지의 글로벌 오퍼레이션에 바로 투입 가능

## 작성 지시사항

아래 항목들을 포함한 자소서를 **한국어**로 작성하라. 필요시 주요 키워드는 영어 병기 가능.

### 1. 지원동기 (400~600자)

- 금호석유화학에서 화학 B2B 트레이딩을 하며 느낀 한계점을 솔직하게 서술
  - 예: 전통 산업의 느린 의사결정, 직관에 의존하는 영업 문화, 데이터 활용 부족
- 트릿지를 알게 된 계기와 충격/인사이트
  - "트레이딩을 데이터로 재정의하는 회사"라는 발견의 순간
- 왜 지금 트릿지여야 하는가: 시리즈 D 단계에서 글로벌 스케일업에 기여할 수 있는 타이밍

### 2. 직무 적합성: B2B 세일즈 + 데이터 분석 융합 (500~700자)

- 화학 B2B 트레이딩에서의 구체적 성과를 **숫자와 사례**로 제시
  - 바이어 발굴 건수, 계약 체결 규모, 신규 시장 진입 성공 사례 등
- Python/AI를 영업에 접목한 구체적 사례
  - 예: 시장 가격 데이터 스크래핑 → 가격 협상 전략 수립
  - 예: 바이어 포트폴리오 분석 자동화 → 우선순위 타겟팅
- 이 경험이 트릿지의 데이터 기반 컨설팅 세일즈에 어떻게 직결되는지 연결

### 3. 글로벌 역량 (300~400자)

- 해외 바이어와의 커뮤니케이션 경험 (영어)
- 문화적 차이를 극복한 협상 사례
- 글로벌 공급망 이슈 대응 경험 (물류, 클레임 등)

### 4. 포부 및 비전 (200~300자)

- 트릿지에서 1년, 3년 후 그리고 싶은 모습
- "단순한 세일즈맨을 넘어, 데이터가 무기인 차세대 트레이더"로서의 정체성

## 작성 원칙

1. **Show, don't tell**: "저는 데이터 분석에 능숙합니다" 대신 실제로 Python으로 무엇을 분석했고 어떤 결과를 냈는지 쓴다.
2. **산업을 관통하는 인사이트**: 단순히 "열심히 하겠습니다"가 아니라, 커머디티 트레이딩 산업에 대한 날카로운 통찰을 보여준다.
3. **Tridge-specific**: 트릿지의 웹사이트, 블로그, 뉴스 기사를 검색하여 회사에 대한 이해를 반영한다. (Tridge의 CoC, Vision, 최근 뉴스 등을 자소서에 자연스럽게 녹인다.)
4. **공격적이되 겸손하지 않게**: 시리즈 D 유니콘에 지원하는 만큼, 자신감 있는 톤을 유지한다. "배우겠습니다"보다 "기여하겠습니다"의 비율이 높아야 한다.
5. **한국 대기업 → 스타트업 전환의 설득력**: 왜 안정적인 대기업을 떠나 스타트업에 합류하려 하는지 납득 가능한 스토리를 제시한다.
