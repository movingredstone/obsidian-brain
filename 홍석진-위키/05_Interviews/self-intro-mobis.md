# ✍️ 현대모비스 자기소개서 (최종)

## 1. 지원동기 — 소재 영업에서 OE 영업으로

> *"시황과 원가를 아는 것이 힘이다"*

중동 PO 공급 위기 — 세계 공장 가동률 데이터로 "3개월 내 공급 부족" 보고서 → 고객사 선제 공유 → **톤당 +$600 계약 갱신**

부품 원가의 큰 몫 = 소재(고무·강재·수지). 3년간 소재 시황과 원가 구조를 다룬 경험을 자동차 OE 영업으로 확장.

조향·현가 OE = RFQ→원가분석→견적→협상→개발총괄→양산 프로젝트 전 사이클. 조향(R-MDPS)·현가(전자제어 서스펜션) 전장화 진행 중.

**목표:** 단기(해외공장 OE 체화) → 중기(전동화 플랫폼 글로벌 OE) → 장기(OE 매출 확대 리더)

## 2. 핵심 역량 — 기다리지 않고 먼저 움직이는 영업

**🇮🇳 인도 신시장 개척**
반덤핑 관세 정보 → 통관 10만건 R/Python 전처리 → HS코드 역추적 → Tokenization 정제 → 현지 5곳 발굴 → 신규 거래선 확보

**🇯🇵 일본 물량 2배**
고객 재고·생산계획 선제 확인 → 납기 안정성 강조 → 전년 동월 대비 2배

**🔧 프로세스 개선**
chemSHERPA·ELV 규제대응 반복 지연 → 실시간 소통채널 + 규제대응 DB 구축 → **대응시간 50% 단축**

## 3. 강점과 약점 — 숫자와 구조로 설득

**💪 강점 1: 원가·시황 근거 설득**
브라질 -$30 인하 요구 → 원료 상승 + 경쟁사 셧다운 데이터 → **오히려 가격 인상**

**💪 강점 2: 비효율을 구조로**
감사 1,000건+ 서류 수작업 = 1개월 → AI·VBA 자동화 → **3일**

**🔧 약점: 조향·현가 공학**
화학 비전공→1년 만에 팀 매출 20% 책임자로 성장한 경험으로 극복. 제품 구조·RFQ 기준·원가 항목·개발 프로세스를 같은 방식으로 습득 예정
