# 마케팅 (Marketing)

## 개요
데이터 기반 고객 세분화와 맞춤형 전략 수립을 통해 시장 점유율을 확대하는 직무.

## 지원 회사 (2025)
- [[현대카드]] — 마케팅/상품기획
- [[KT&G]] — 글로벌 마케팅

## 핵심 역량

### 고객 세분화
- 나이지리아·태국·일본 등 다양한 문화권 고객군 특성 분석
- 국가별 결제 방식·구매 결정 요인의 차이 체감 → 맞춤형 접근

### 데이터 기반 마케팅
- 통관자료 10만 건 분석 → 시장 세분화 → 신규 업체 발굴
- "숫자 뒤에 숨은 스토리를 읽어내는 능력"
- 통계학과 편입 → R 프로그래밍, 통계적 추론
- [[customs-data-analysis|통관데이터 분석]]

### 고객 경험 개선
- [[일본-규제대응-프로세스-개선]] → 처리시간 단축, 매출 증가
- 예상치 못한 변화를 기회로 전환

### 글로벌 마케팅 전략
- 러시아·중앙아시아 시장 전문성
- SWOT·STP 분석 → 경쟁사 대비 판매국 27% 부족 파악 → 신시장 개척
