# 프로젝트 관리 (Project Management)

## 개요
전략사업개발과 연계되어 아이디어를 현실화하고, 기술적 검토와 사업성 분석을 바탕으로 다양한 부서 간 협업을 이끌어가는 직무.

## 지원 회사 (2025)
- [[현대모비스]] — 전략사업개발/프로젝트 관리

## 핵심 역량

### 프로젝트 리더십
- [[키토산-필터-개발]]: 아이디어 구체화 → 실험 → 사업 전략 수립 → 외부 발표까지 전반 주도
- 팀 리더로서 역할 분담, 연구기관 검증 의뢰, SWOT·4P 전략 자료 체계적 준비
- 외부 경진대회 '창의상' 수상

### 이해관계자 조율
- [[일본-규제대응-프로세스-개선]]:
  - 품보팀 vs 영업팀 간 불신 해소
  - 규제 DB 구축 + 전용 소통 채널 → 워크플로우 설계
  - 6개월 내 수주 대응 시간 30% 단축, 매출 15% 증가

### 실행 중심 사고
- "위기 속에서 포기하지 않고, 데이터를 기반으로 문제를 진단하고 협업 구조를 설계"
- 고객사에 내부 대응 구조 투명하게 설명 → 신뢰 회복
