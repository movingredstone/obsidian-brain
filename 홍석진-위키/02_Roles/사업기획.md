# 사업기획 (Business Planning)

## 개요
글로벌 정세 변화, 시장 동향, 기술 트렌드를 종합 분석해 기업의 미래 성장 동력을 발굴하고 실행 가능한 로드맵으로 구체화하는 직무. 사업 포트폴리오 전략을 통해 지속가능한 경쟁우위를 확보.

- [[GS칼텍스]] — 사업기획/해외영업
- [[현대카드]] — 마케팅/상품기획

## 핵심 역량

### 질문에서 시작하는 기획
- "모든 기획은 왜라는 질문에서 시작된다"
- "왜 지연되는가?" → 일본 규제 대응 프로세스 재설계
- [[일본-규제대응-프로세스-개선]]

### 거시 트렌드 분석
- [[러시아-우크라이나-전쟁-예측]] → 글로벌 공급망 재편·에너지 시장 변화 선제 분석
- 중장기 전략 수립 시 외부 리스크 요인 사전 식별 → 대응 전략 마련

### 정량적 근거 기반 전략
- 통관자료 10만 건 분석 → 수출입 흐름 파악 → 영업전략 수립
- 통계학과 편입 → 분석 역량 이론적 강화
- 공공 API 데이터 ETL → 상권분석 전략 수립
- [[customs-data-analysis|통관데이터 분석]]

### 협업 통한 목표 달성
- [[kci-논문-등재-인도-방산]]: 다학제 팀 리드 → 주 2회 정기 미팅 → 역할 분담 → 공모전 1등
- 각자 강점 인정 + 공통 목표로 결집시키는 조정 능력

## 사업기획 직무 철학
- "불확실성이 커지는 경영환경에서 리스크를 기회로 전환"
- "전사 차원의 리소스 배분을 최적화하는 전략적 사고"
- "학제간 융합이 새로운 시너지를 창출"
