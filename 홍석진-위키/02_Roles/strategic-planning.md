# 전략기획 (Strategic Planning)

## 개요
MI(Market Intelligence), 대내외 정책들을 연결해 새로운 인사이트를 도출하고, 기업의 중장기 방향성을 설계하는 직무. "점을 찾아 선을 잇는 사람".

## 지원 회사 (2025)
- [[GS에너지]] — 전략기획

## 핵심 역량

### 통합적 분석
- 정책·원료·수출입 데이터를 통합한 손익 기반 전략 자료 제공
- PPG 제품 해외영업·시장 분석 경험 → 전략 수립 기여

### 질문 기반 전략
- "한국 방산업체가 어떻게 하면 인도 시장에 진출할 수 있을까?" → KCI 논문 등재
- "어떻게 하면 수익성을 확보할 수 있을까?" → 무역조건별 순매출 자동 계산 시스템 도입
- [[kci-논문-등재-인도-방산]]

### 아이디어 실현
- [[키토산-필터-개발]]: 러시아 수질 문제 → 비즈니스 모델 구체화
- 시제품 개발·원가분석·손익 추정 → "비용이 수익보다 컸기에 실제 사업화는 불가, 하지만 시장 분석은 훌륭"

### 데이터 분석
- 통관자료 10만 건 Tokenization/Lemmatization → 수요 파악
- 통계학과 진학 → R·엑셀 활용 분석
- [[customs-data-analysis|통관데이터 분석]]

### 팀 성과 기여
- [[일본-규제대응-프로세스-개선]]: 규제별 DB + 전용 소통 채널
- "원활한 소통이 성과에 얼마나 중요한지"
