# 해외영업 역량

## 정의
B2B/B2C 제품의 해외 시장 판매, 고객 관리, 시장 개척을 총괄하는 직무. 글로벌 비즈니스의 최전선.

## 핵심 역량 요소

### 1. 기본기 (Fundamentals)
- "최고의 영업 기술은 고객의 편안함을 끌어내는 것"
- 제품, 수량, 도착일 등 기본 정보 확인의 중요성
- [[도료-배송-기본기-경험]] 참조
- 기본에 충실 → 업무 효율화 + 고객 신뢰

### 2. 시장 분석력
- 권역별 생산 능력·수요 변화 분석 → 맞춤형 전략
- 원재료 가격 모니터링 → 제품 판가 반영
- 경쟁사 분석, SWOT/STP 전략 수립
- [[인도-PPG-시장개척]]: 인도 시장 진출 사례

### 3. 데이터 리터러시
- 통관 자료 분석 → end user 식별, distributor 발굴
- [[customs-data-analysis|통관데이터 분석]]: 10만 건 데이터 처리
- R, 엑셀, Tokenization, Lemmatization 기술
- 통계 데이터과학과 편입 → 지속적 학습

### 4. 정책·규제 이해
- 국제 정세, 반덤핑 관세, 환경 규제 등이 영업에 미치는 영향
- "제품 판매에서 가장 중요한 것 = 상품 가격 + 규제"
- [[인도-PPG-시장개척]]: 중국산 반덤핑 관세 기회 포착

### 5. 소통과 관계 구축
- 클라이언트 편안함 유도
- 국가별 다른 목소리 청취 → 제품 반영
- 현지 유통업자·distributor 발굴 및 관계 유지
