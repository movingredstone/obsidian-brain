# 홍보·마케팅 역량

## 정의
기업/제품의 이미지 제고, 판촉 활동, 콘텐츠 기획을 통한 시장 커뮤니케이션 직무.

## 핵심 역량 요소

### 1. 타겟 분석 및 맞춤형 전략
- 연령대, 관심사 등 타겟 특성 분석
- 타겟에 맞는 홍보 방식 선정
- [[현대케피코|현대케피코]] 예: 투자자 50~60대 → '담백한' 홍보물
- 결과: "연령대를 잘 분석했다" 피드백

### 2. Sales Point 도출
- 제품의 차별화 포인트 분석
- SWOT 분석, STP 전략 활용
- 경쟁사 대비 강점/약점 파악 → 홍보 포인트 발굴
- [[인도-PPG-시장개척]]: 경쟁사 대비 판매국 27% 적음 → 시장 개척 필요성 인식

### 3. 콘텐츠 기획 및 제작
- 홍보물(팜플릿 등) 기획·제작
- 외부 업체(사진작가, 디자이너) 섭외
- [[키토산-필터-프로젝트]]: 시제품 홍보 콘텐츠 제작

### 4. 학보사 콘텐츠 경험
- 신문 판형 변경(Broadsheet → Tabloid) → 휴대성 증대
- 대학생 니즈 분석 → 성공스토리 콘텐츠 기획
- 조회수 30% 증가, 주 콘텐츠로 정착
