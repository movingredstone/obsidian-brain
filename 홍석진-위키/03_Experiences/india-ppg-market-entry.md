# 인도 PPG 시장 개척

## Situation (상황)
- **직무**: PPG(폴리프로필렌 글리콜) 해외영업 담당
- **맥락**: 회사에 인도 거래선이 전무한 상태
- **기회**: 인도가 중국산 PPG에 대해 **반덤핑 관세**를 부과한다는 정보 입수
- **분석**: 기존 기술·고객만족 세미나에서 SWOT/STP 분석 결과 경쟁사 대비 판매국이 **27% 적음** → 신규 시장 개척 필요

## Task (과제)
- 인도 시장 진출을 위한 유통업자(distributor) 발굴
- 판매량 예측을 위한 통관 자료 확보·분석
- 현지 바이어 컨택 및 관계 구축

## Action (행동)

### 1. 정보 공유 및 전략 수립
- 반덤핑 관세 정보를 팀원에게 공유 → 인도 유통업자 탐색 결정
- 상사를 통해 인도·미국 통관 자료 15,000건 입수

### 2. 데이터 분석 (난관 극복)
- **문제**: HS Code 불일치로 어떤 업체가 경쟁사 제품을 구매하는지 확인 불가
- 불필요한 데이터와 복잡한 구조로 분석에 4일 소요
- 데이터 가공 능력 부족 인식
- **해결**:
  - R·엑셀 활용 데이터 분석 기술 학습
  - Tokenization, Lemmatization 등 텍스트 프로세싱 기술 적용
  - 현지 distributor 분류 및 end user 리스트업

### 3. 지속적 학습
- 데이터의 중요성 깨닫고 **통계 데이터과학과 3학년 편입**
- 온라인 강의 수강, 관련 서적 참고, 실습 반복
- 주경야독(직장+학업 병행)

### 4. 현지 컨택
- 인도 유통업자 발굴·연락
- 한 업체의 반응: "너희는 인도 시장 내 Dow와 같은 기업들과 경쟁도 되지 않는다"
- 포기하지 않고 지속적 컨택 → 현지 업체 발굴 성공

## Result (결과)
- 인도 유통업자와 연락 성공
- 중국의 빈자리를 자사 제품으로 채울 기회 확보
- "그 말이 틀렸다는 것을 증명했다"
- **핵심 교훈**: "모른다면 배우고, 방법을 찾아야 한다"
