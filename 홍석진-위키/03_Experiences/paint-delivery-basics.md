# 도료 배송 기본기 경험

## Situation (상황)
- **시기**: 입사 초기 (현 직장, PPG 영업)
- **맥락**: 도료 공장에서 제품 배송 가능 여부 확인 요청 접수

## Task (과제)
- 클라이언트 요청에 신속히 응대
- 배송 가능 여부 확인 후 회신

## Action (행동) — 실수와 교정

### 실수
- 아무것도 모르는 상태에서 즉시 배송기사님에게 전화
- "배송 가능한가요?"만 물어봄
- 기사님 답변: **"제품, 수량, 도착일이 뭐죠?"**
- 기본 정보도 모른 채 클라이언트에게 "알겠습니다"라고 응답

### 깨달음과 교정
- 기본적인 사항(제품명, 수량, 도착일)도 묻지 않은 사실에 혼남
- 재차 클라이언트에게 전화 → 기본 정보 확인 → 정상 주문 처리
- **추가 프로세스 발생 + 클라이언트 불편 초래** → 반성

## Result (결과)

### 체득한 교훈
- **"최고의 영업 기술은 고객의 편안함을 끌어내는 것"**
- **"이 편안함은 '기본기'에서 나온다"**
- 기본에 충실하면: 업무 프로세스 효율화 + 고객 불편 방지

### 파급 효과
- 이후 모든 업무에서 **사전 조사를 꼼꼼하게** 하는 습관
- **한 번 더 확인**하는 습관 형성
- 다음 단계를 예측하며 일하는 능력 향상
- 예: 데이터 분석 시 사전에 자료 용도 파악 → 송하인·제품 카테고리 추가 → **3일 내 데이터 가공** (기존 4일 → 3일로 단축)

## STAR 요약
| 요소 | 내용 |
|---|---|
| **S** | 입사 초기, 제품 배송 가능 여부 확인 요청 접수 |
| **T** | 클라이언트 요청에 정확히 응대 |
| **A** | 기본 정보 확인 없이 응답(실수) → 재확인 후 정상 처리(교정) |
| **R** | 기본의 중요성 체득, 모든 업무에 사전 확인 습관화 |
