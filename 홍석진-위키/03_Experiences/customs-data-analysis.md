# 통관데이터 10만건 분석

## STAR 요약

### Situation (상황)
인도·미국 시장의 수출입 흐름을 분석해야 하는 과제. 통관자료는 HS CODE가 국가별로 상이하고 DB가 방대하여(10만 건 이상) 수작업 시 2주~1개월 이상 소요. 대부분이 불가능하다고 판단하는 작업.

### Task (과제)
방대한 통관자료를 효율적으로 전처리·분석하여 실제 영업 전략 수립에 활용 가능한 인사이트 도출.

### Action (행동)
1. **통계학과 편입**: 데이터 분석의 중요성을 느껴 방송통신대 통계학과 3학년 편입
2. **도구 학습**: R 프로그래밍, Python 기초 학습 (직장+학업 병행)
3. **데이터 전처리**: Tokenization, Lemmatization 기법 적용 → 제품명·송하인·수하인 데이터 정제·분류
4. **분석 기준 설계**: 단순 데이터 정리를 넘어 분석 기준 정의 및 보고 체계 설계
5. **활용**: 분석 결과 → 실제 수출 전략 수립에 기여

### Result (결과)
- 수작업 1개월 → **5일** 만에 처리 완료 (R·엑셀 매크로)
- 신규 유통업체 **5곳** 발굴 및 컨택 성공
- 수출입 흐름 파악 → 영업 전략 수립에 실질적 기여
- 통계학과 편입으로 이론적 기반도 강화

## 채용 지원서 내 변형
- **10만 건**: 한화시스템, 한화글로벌, 현대모비스, 현대오일뱅크, 현대마린솔루션, GS에너지, KT&G 등
- **2만 건 → 3일**: 한화에어로스페이스, LG유플러스 (68곳 업체 확보, 3곳 판매 성사)
- **3일 만에 매크로 개발**: 현대오일뱅크 (5곳 발굴)
