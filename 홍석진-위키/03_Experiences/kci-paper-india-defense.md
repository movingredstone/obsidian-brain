# KCI 논문 등재 - 인도 방산 시장 진출 전략

## STAR 요약

### Situation (상황)
학부생 시절, "한국 방산업체는 왜 러시아처럼 인도 시장에 진출할 수 없을까?"라는 질문에서 시작. 인도는 세계 2위 무기 수입국으로, 냉전시대부터 러시아가 방산 시장을 지배. 그러나 모디 정부의 '메이크 인 인디아' 정책과 지정학적 긴장으로 서방 무기 체계로 전환 중 → 한국 방산업체에 기회.

### Task (과제)
지도교수 없이 학부생이 단독으로 KCI 등재지에 논문을 게재하는 것 — 사실상 불가능한 목표.

### Action (행동)
1. **우회 전략**: 논문 공모전 수상 시 학술지 게재 기회 확보 가능 → 공모전 특성에 맞춰 연구 주제 심화
2. **연구 방향 설정**: 인도-러시아 군사협력의 구조적 모순에 주목 → 인도의 러시아산 무기 의존도와 공급 불안정성 리스크 분석
3. **다학제 협업**: 국제학(본인) + 경영학 + 정치외교학 팀원과 역할 분담
4. **1차 자료 활용**: 영어·러시아어 1차 자료까지 분석 → 기존 연구와 차별화
5. **체계적 연구**: 3개월간 주 2회 정기 미팅, 인도 방위산업 정책·법규·역사적 맥락 심층 분석
6. **다언어 자료**: 러시아 국방부·인도 의회 자료 교차 분석 → 양국 군사협력 한계점 규명

### Result (결과)
- 논문 공모전 **1등** 수상
- 심사평: "학부생 수준을 뛰어넘는 전문성과 독창성", "풍부한 근거자료와 연구 목적이 괄목할만함"
- KCI 학술지 최종 등재 성공
- "논문 공모전이라는 새로운 기회를 포착한 순발력 + 연구 전략 즉시 수정한 적응력이 핵심"

## 핵심 교훈
- "못할 것 같다고 생각하면 정말 못하게 되고, '할 수 있다!'라는 생각을 가지면 결국 성공"
- "해보지 않고서는 아무것도 이뤄낼 수 없다"
- "다른 전문성을 가진 사람들과 협업할 때 각자 강점을 인정하면서 공통 목표로 결집시키는 조정 능력이 핵심"
- "불가능해 보이는 상황에서도 기회를 포착하고 실행을 통해 돌파"
