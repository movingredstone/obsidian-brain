# 키토산 필터 개발

## STAR 요약

### Situation (상황)
외교부 주관 사업 아이디어 공모전/창업경진대회 참가. 러시아의 수질 문제(노후 수도관) 해결을 위한 아이디어가 필요했고, 버려지는 대게 껍데기를 활용하는 방안을 제안함. 러시아가 킹크랩·대게 원산지인 점에 착안한 현지화 전략.

### Task (과제)
대게 껍데기에서 키토산을 추출해 음수 필터를 개발하는 프로젝트 주도. 초기 팀원들은 기술적 실현 가능성과 투자 유치 가능성을 의심하며 반대.

### Action (행동)
1. **데이터 확보**: 용돈을 투자하여 시제품 직접 생산, 연구기관에 효능 검증 의뢰 → 유해물질 제거 효율 98% 데이터 확보
2. **사업성 분석**: SWOT 분석, 4P 마케팅 전략 작성 → 시장 규모, 경쟁사 현황, 수익 모델 정량적 제시
3. **팀 설득**: 단계적 개발 로드맵 수립 → 팀 전체 합의 도출
4. **시연**: 개선된 시제품으로 투자자 대상 시연 성공 → 실제 오염수 정화 과정 실증
5. **차별화**: 러시아 킹크랩·대게 원산지 특성 활용 → 현지 원료 조달 가능성 + 환경 친화적 특성 부각

### Result (결과)
- 시제품 개발 완료 및 투자자 시연 성공
- 외부 경진대회 '창의상' 수상
- 높은 제조원가로 양산 실패 → 그러나 실패보다 배움이 큼
- "안 된다는 선입견에 도전하고, 구체적 근거와 체계적 계획으로 불가능을 현실로 만드는 경험"

## 핵심 교훈
- "누군가 '안 된다'라고 할 때 포기하면 정말 안 되는 것. 구체적 근거와 체계적 계획으로 불가능이 현실이 된다"
- "시도조차 안 한다는 것 자체가 실패"
- "누군가를 설득할 때 가장 중요한 건 데이터"
- "생각하는 것에서 그치지 않고 실행하는 것이 중요"
