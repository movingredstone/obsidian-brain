# 📝 이력서 자동 생성기

> 위키 데이터로 이력서를 자동 생성합니다.
> Hermes Agent가 템플릿에 위키 내용을 채워넣습니다.

## 📄 사용법

### 방법 1: 명령어로 생성
```
Hermes에게: "내 해외영업 이력서 생성해줘"
→ 위키의 경험/스킬을 자동으로 조합해서 이력서 출력
```

### 방법 2: 회사 맞춤형
```
"현대차 지원용 이력서 만들어줘"
→ 현대자동차에 최적화된 이력서 생성
```

### 방법 3: 크론잡 자동 업데이트
- 주 1회 위키 변경사항 반영하여 이력서 갱신
