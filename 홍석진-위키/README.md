# 🧠 홍석진 Career Wiki

> Obsidian 기반 LLM 최적화 커리어 지식 베이스
> Claude · DeepSeek · Codex · RAG 파이프라인 호환

## 📂 구조

```
00_Indexes/          → 회사별, 직무별, 경험별 내비게이션
01_Companies/        → 지원 회사별 분석 (현대, 한화, GS 등 37개사)
02_Roles/            → 직무별 역량 정리 (해외영업, 사업기획 등)
03_Experiences/      → STAR 스토리, 경험 블록
04_Projects/         → 개인 프로젝트 (BOOK ME UP 등)
05_Interviews/       → 면접 질문/답변, PT 자료
06_Resumes/          → 이력서 버전 관리
07_Career_Descriptions/ → 자기소개 통합
08_Certifications/   → 자격증
09_Education/        → 학력
10_Knowledge_Graph/  → 스킬, 산업, 키워드 그래프
11_Job_Tracker/      → 구직 트래커
12_Resume_Generator/ → 이력서 자동 생성기
13_Career_Strategy/  → 커리어 전략, 지원 전략
raw/                 → 원본 PDF (수정 불가)
SCHEMA.md            → 위키 규칙
index.md             → 전체 카탈로그
log.md               → 작업 로그
```

## 🚀 시작하기

1. Obsidian에서 이 폴더를 Vault로 열기
2. `00_Indexes/회사별_인덱스.md`에서 시작
3. 그래프 뷰로 전체 지식 네트워크 탐색

## 🔍 LLM 검색 최적화

- 모든 페이지에 구조화된 YAML frontmatter
- `[[wikilinks]]`로 고밀도 상호 연결
- 태그 taxonomy 기반 시맨틱 검색 가능
- STAR 포맷 경험 블록 재사용 가능

## 🛡️ 원본 보존

`raw/` 폴더의 모든 PDF는 원본 그대로 보존됩니다.
절대 수정되거나 삭제되지 않습니다.
