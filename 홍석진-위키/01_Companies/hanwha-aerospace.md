# 한화에어로스페이스

## 개요
한화에어로스페이스는 대한민국을 대표하는 방산업체로 육해공을 아우르는 사업을 전개. K9 자주포, 항공기 엔진, 우주 발사체 등 첨단 방산·항공 제품 포트폴리오 보유.

## 2025 지원 내역
- **지원 직무**: 사업관리 (해외시장 조사, 수출현황 관리, 해외영업 수주 활동 지원)
- **지원 시기**: 2025년

## 지원 동기
군에서 해외 무기체계 분석 보직 수행 → 방위산업 열정 발생. 제대 후 한국 방위산업체가 인도시장 진출 방안을 연구하는 논문 KCI 등재. 비전: "Defense Total Solutions Provider" — 제품 공급을 넘어 지정학적 분석과 솔루션 제공.

## 직무 역량

### 경험과 데이터 리터러시
- 석유화학 기업 해외영업, 수출현황 관리, 채널 관리, 시장 조사 수행
- 통계학과 편입 → 데이터 리터러시 함양
- 통관자료 2만 건 이상 Tokenization 전처리 → 3일 내 68곳 업체 확보, 3곳 판매 성사
- 데이터 리터러시 = 단순 해석이 아닌 인사이트 창출

### 문제 해결형 주인의식
- 일본 판매 확대 목표 → 전년 동기 대비 매출 5% 상승 달성
- "왜?"라는 질문으로 소통 부재 원인 규명
- 고객 요청 양식 설계 + 협업툴 핫라인 구축
- [[일본-규제대응-프로세스-개선]]

### 융합적 사고
- 대학: 에너지, 경제, 방위산업 연구 학회/동아리 활동
- 러-우 전쟁 발발 전(2021년) 갈등 가능성 예측 → 논문 대회 입상
- 대게 껍데기 키토산 음수 필터 개발 → SWOT·4P 전략 제시
- 배움을 연결해 새로운 가치 창출

### 변화 수용성
- 직장+학업 병행: BS4 패키지로 뉴스 크롤링·여론 조사
- 공공 API 데이터 ETL 처리 → 상권분석 전략 수립
- 원가회계 독학, 경영학 복수전공
