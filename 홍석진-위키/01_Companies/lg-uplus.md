# LG유플러스

## 개요
LG유플러스는 LG그룹의 통신사로, 유무선 통신 서비스를 제공하며 데이터 기반 사업 혁신을 추진 중이다.

## 2025 지원 내역
- **지원 직무**: 사업기획
- **지원 시기**: 2025년

## 직무 역량

### 학문적 분석력
- **비즈니스협상론(A+)**: 게임이론·블러핑 등 협상 기술 학습, 롤플레잉에서 유가·SCFI 지수 상관관계 근거 블러핑 성공
- **국제정치경제연구(A+)**: 거시경제 분석
- **파이썬 컴퓨팅(B+)**: 데이터 기반 의사결정 필요성 → 통계학과 편입 계기, 공공 API 데이터 ETL → 상권분석 전략 수립

### 데이터 리터러시
- 인도 판매 확대 목표 → 통관자료 2만 건 Tokenization 처리
- 상부 3일 내 처리 요구 → 파이썬 활용 전처리 → 68곳 업체 확보, 3곳 판매 성사
- [[customs-data-analysis|통관데이터 분석]]

### 프로세스 혁신
- [[일본-규제대응-프로세스-개선]]: chemSHERPA 대응
- "왜?"라는 질문 → 소통 부재 원인 규명
- 고객 요청 양식 설계 + 협업툴 핫라인 구축 → 전년 동기 대비 매출 5% 상승
