# LS MnM

## 개요
LS그룹 비철금속 제련 계열사(Minerals & Metals). 구리, 금 등 제련 사업.

## 2024년 지원
- **지원 분야**: 해외영업 / 사업개발
- **지원 시기**: 2024년

## 자기소개: "우당탕탕"
- 무언가 요란스럽게 부딪치며 문제 해결하는 모습
- 미숙하지만 이리저리 부딪치며 배우는 자세
- "모른다면 배우고, 방법을 찾아야 한다"

## 강조한 핵심 역량

### 1. 지속적 학습과 문제 해결
- 인도 중국산 PPG 반덤핑 관세 → 유통업자 발굴 미션
- HS Code 불일치로 통관자료 확보 어려움
- 15,000개 통관자료 분석, 데이터 가공 능력 부족 인식 → 통계 데이터학과 편입
- Tokenization, Lemmatization → 현지 유통업체 분류·컨택
- "Dow와 경쟁도 안 된다"는 반응에도 현지 업체 발굴 성공

### 2. 분석력과 예측력
- [[전쟁-예측-논문]]: 2021년 우크라이나-러시아 전쟁 예측
- 우크라이나 탈러시아 기조, 지정학적 위치('러시아의 목')
- 논문공모전 입상
- 방위산업체 인도 진출 방안 KCI 논문 등재
- 한국-인도 방위산업 법·제도 비교 분석

### 3. 협업과 가치 창출
- [[키토산-필터-프로젝트]]: 러시아 수질 문제 해결 아이디어
- 러시아 정수기보다 생수 선호 → 대체 제품
- 게 껍데기 키토산 필터 시제품 생산 → 입상
