# CJ올리브영

## 개요
CJ올리브영은 국내 1위 H&B 스토어. K뷰티 글로벌 B2B 영업(Western 권역) 직무에 지원.

## 2026 지원 내역
- **지원 직무**: 글로벌 B2B 영업 (Western — 유럽/동유럽)
- **지원 경로**: LinkedIn
- **지원 시기**: 2026년 6월

## 지원 동기
> "저는 LEADER가 아닌 READER가 되고 싶습니다"

국가마다 산업 구조, 유통 채널, 고객 우선순위, 경쟁 구도가 다르므로 같은 제품이라도 접근 방식을 달리해야 함. 시장을 먼저 읽고 거래처가 필요로 하는 것을 선제적으로 제안하는 방식이 실제 성과를 낸다는 것을 체험. K뷰티가 글로벌 시장에서 스포트라이트 받는 지금, 올리브영과 함께하고 싶다는 포부.

## 직무 역량

### 시장을 읽는 영업 (Reader)
- 일본 시장: 이란-미국 전쟁 → 공급망 혼란 → 소싱 이원화·삼원화 니즈 포착 → 물량 100% 확대
- 인도 시장: 직접 수출 실적 無 → 10만 건 통관 데이터 → 파이썬 전처리 → 가격 기준 및 우선 검토 업체 도출
- 태국 HS CODE 차이(3907.90 vs 3907.20) 사례 → 디테일이 리드타임에 미치는 영향 체득

### 점을 찾아 선을 잇는 협상
- 브라질 ITAPOA: 30달러 인하 요청 → 美 경쟁사 화재 + 원재료 상승 → 가격 방어
- 직전 체결가 2,350달러 → 시장 분석 → 2,370달러 계약
- 글로벌 메이커 D사: 전쟁 장기화 예측 → 타이밍 조율 → 톤당 600달러 인상
- [[가격협상-전략]]

### 수출입 실무 + 자동화
- 발주, 재고 수급, 출고, 정산까지 전 프로세스 직접 관리
- 판가 운영, 덤핑 모니터링, PO 가격 동향, 추정손익 집계
- 외국환거래 감사 대응: 5년치 1,000건+ → AI+VBA → 1개월→3일
- [[customs-data-analysis|통관데이터 분석]]

### 데이터 리터러시
- 수출입 동향 추적, 경쟁사 판매 흐름 파악, 추정손익 집계
- 통관자료 10만 건 파이썬 전처리 경험
- 데이터와 실무를 연결하는 역량 → 의사결정 근거로 활용

### 언어·문화 커뮤니케이션
- **러시아어** 구사 (OPIc Advanced Low) — 폴란드·체코·헝가리 등 동유럽 비즈니스에 실질적 강점
- 일본(신뢰·공급 안정성), 남미(가격 논리·수익성), 인도(데이터 기반 분석) → 문화별 맞춤 접근
- 다국가 거래처 관리 역량 → Western 권역 업무에 직결

## 주요 성과
1. 일본 시장 동년 동월 대비 물량 100% 확대
2. 브라질 ITAPOA 가격 방어
3. D사 톤당 600달러 인상 계약 체결
4. 인도 시장 데이터 기반 진입 분석
5. 감사 대응 AI·VBA 자동화 (1개월→3일)
