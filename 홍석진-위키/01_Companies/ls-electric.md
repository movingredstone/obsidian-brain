# LS ELECTRIC

## 개요
LS ELECTRIC은 전력기기·자동화 솔루션 기업으로, 권역별 산업 환경에 맞는 사업 기회를 직접 만들어 가는 회사. 유럽 해외사업 직무에 지원.

## 2026 지원 내역
- **지원 직무**: 유럽 해외사업 (신규 거래선 발굴, Key Account 관리)
- **지원 시기**: 2026년 6월

## 지원 동기
> "기회는 기다리는 것이 아니라, 잡는 것입니다"

석유화학 해외영업에서 스팟 거래보다 고객과 함께 사업을 키워가는 방식에 더 큰 가능성을 느낌. 전력기기·자동화 솔루션처럼 산업 인프라와 직결되는 분야에서 Key Account를 발굴하고 장기 파트너십을 구축하는 일을 하고 싶다는 방향성. LS ELECTRIC은 권역별 산업 환경에 맞는 사업 기회를 직접 만들어 가는 회사라는 점에서 가장 잘 맞닿아 있다고 판단.

## 직무 역량

### 시장을 읽는 능력 (Reader, not Leader)
- 해외영업 = 제품 소개가 아닌 시장 상황, 경쟁 구도, 고객 수용 범위, 내부 손익을 함께 고려해 더 나은 조건을 만드는 일
- 판가·덤핑 모니터링, PO 가격 동향, 추정손익 집계 수행
- 다양한 출처 정보를 하나의 시장 흐름으로 연결 → 협상 우위로 전환

### 시장 기반 협상력
- 브라질 ITAPOA향: 국내 상사 30달러 인하 요청 → 미국 경쟁사 화재 + 원재료 상승 파악 → 가격 네고 없이 기존 조건 유지
- 직전 체결가 2,350달러 → 시장 흐름 + 고객 수용 범위 분석 → 2,370달러 계약 체결
- 글로벌 메이커 D사: 이란-미국 전쟁 장기화 예측 → 협상 타이밍 지연 → 톤당 600달러 높은 조건 체결
- [[가격협상-전략]]

### 데이터 기반 시장 개척
- 인도 시장: 직접 수출 실적 無 → 10만 건 통관 데이터 확보 → 파이썬 전처리
- HS CODE 필터링, 주요 수입업체·거래 빈도·평균 시장가 도출
- 막연한 시장을 데이터 기반으로 해석 → 내부 활용 가격 기준 및 우선 검토 업체군 확보
- [[customs-data-analysis|통관데이터 분석]]

### 운영 효율화
- 외국환거래 감사 대응: 5년치 1,000건+ CI/PL/BL → AI+VBA 자동화 → 1개월→3일 단축
- 반복 업무를 구조적으로 개선하고 새로운 도구를 빠르게 학습·적용

## 주요 성과
1. 일본 시장 42톤→84톤 확대 (이란-미국 전쟁 공급망 교란 포착)
2. 브라질 ITAPOA 가격 방어 (30달러 인하 요청 방어)
3. D사 톤당 600달러 인상 계약 체결
4. 인도 시장 데이터 기반 진입 분석
5. 감사 대응 AI·VBA 자동화 (1개월→3일)
