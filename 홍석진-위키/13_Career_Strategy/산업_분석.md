# 🏭 산업 분석 — 어디로 가야 하는가

## 화학/소재 산업 ⭐⭐⭐⭐⭐
**최적 매칭도: 95%**

- PPG 경험 직접 활용 가능
- B2B 중심 → 가격협상·공급망 경험 살림
- 글로벌화 심화 → 해외영업/BD 수요 증가
- 추천: BASF, Dow, SABIC, 한화토탈에너지스, 금호석유(경쟁사)

## 에너지/신재생 ⭐⭐⭐⭐
**매칭도: 80%**

- 빠르게 성장하는 산업
- Commercial/BD 역할 다수
- 추천: Shell, TotalEnergies, 한화솔루션, GS에너지

## 산업재/B2B 제조 ⭐⭐⭐⭐
**매칭도: 85%**

- B2B 영업 경험 직결
- 글로벌 공급망 연계 포지션
- 추천: Siemens, GE, Honeywell, 현대모비스

## Mobility/자동차 ⭐⭐⭐
**매칭도: 70%**

- 전동화 전환기 → BD 기회
- 부품사 글로벌 영업 수요
- 추천: 현대모비스, BorgWarner, Continental

## 소비재/B2C ⭐⭐
**매칭도: 40%**

- B2B 경험 활용도 낮음
- 마케팅 중심 → 전략성 제한
- **비추천**
